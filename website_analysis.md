- Logo: "Homestay Bumina Eenk" dengan ikon rumah

- Gambar besar: Seseorang di kebun teh dengan kabut

- Judul utama: "Penginapan di Taman Langit Pangalengan bandung"

- Konten: Deskripsi lengkap tentang homestay

### 5. Features Section

### 7. Testimonials Section

## Color Palette
- Primary: Navy blue (#2D3748 atau similar)
- Secondary: Hijau muda/mint untuk header
- Accent: Biru untuk CTA buttons
- Background: Putih/off-white
- Text: Dark gray/navy

## Typography
- Headings: Bold, sans-serif
- Body text: Regular, readable
- Hierarchy jelas dengan ukuran berbeda

## Layout Patterns
- Full-width sections
- Container dengan max-width
- Grid layout untuk features
- Split layout untuk hero dan about
- Card-based design untuk testimonials

## Key Images Needed
1. Logo homestay

3. Poster grafis "HOMESTAY BUMINA GRATIS EENK"
4. Gambar kebun teh panorama
5. Foto homestay/villa
6. Foto grup tamu
7. Foto pemandangan gunung
8. Foto fasilitas interior

## Responsive Considerations
- Mobile-first approach
- Stack layout untuk mobile
- Collapsible navigation
- Optimized images
- Touch-friendly buttons
